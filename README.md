# StudySync 🎓

**StudySync** is a productivity-focused mobile application built with **Flutter** and **Dart**. Developed as part of a **Human-Computer Interaction (HCI)** course, this app centralizes essential student tools into one intuitive interface to reduce cognitive load and improve time management.

---

### 🌟 Key Features
* **Smart To-Do List:** A streamlined task manager to categorize and prioritize academic responsibilities.
* **Custom Study Timer:** An integrated focus timer (Pomodoro style) to help maintain deep work sessions.
* **HCI-Focused Design:** Prioritizes "Visibility of System Status" and "User Control," ensuring students can navigate effortlessly during high-stress study sessions.
* **All-in-One Dashboard:** Quick access to various study tools to minimize app-switching and digital distractions.

---

### 🧠 HCI & UX Principles Applied
This project was built with a strong emphasis on user-centered design:
* **Minimalist Aesthetic:** A "clutter-free" UI designed specifically to avoid overwhelming users while they are studying.
* **Consistency and Standards:** Follows Material Design principles to provide a familiar and predictable experience.
* **Error Prevention:** Intuitive input fields and clear navigation to prevent user frustration.

### 🚀 Getting Started

#### Prerequisites
* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* Android Studio / VS Code with Flutter extension
* An emulator or physical device

#### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/knndy-Mr/StudySync.git](https://github.com/knndy-Mr/StudySync.git)
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the application:**
    ```bash
    flutter run
    ```
